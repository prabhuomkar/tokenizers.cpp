# tokenizers.cpp
Tokenizers in C++

## Features
- Easy to use tokenizers library in C++ based on HuggingFace's `tokenizer.json` config.
- Designed for production applications e.g. [Android](android/), [iOS](ios/), [ggml]()

## Examples
- [LibTorch](./examples/libtorch-bert/)
- [ggml]()
- [Android App](./android/)
- [iOS App](./ios/)
